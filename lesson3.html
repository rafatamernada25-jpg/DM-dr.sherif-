Companding & Takeaways

Nyquist-Shannon Sampling Theorem

Foundational rule in signal processing:
A band-limited continuous-time signal can be perfectly reconstructed from its samples if sampled at a rate greater than or equal to twice the highest frequency component.
F_s ≥ 2 × F_max
Core meaning: Sample fast enough (at least twice the highest frequency) → perfect reconstruction is theoretically possible

Nyquist Rate

The minimum theoretical sampling frequency required for perfect reconstruction without information loss.
Nyquist Rate = 2 × F_max
Practical note: Real systems usually use slightly higher sampling rates (e.g. 2.2–2.5×) to account for imperfect filters.

Sampling Process

Conversion of continuous-time signal x(t) into discrete-time signal x[n] by taking periodic snapshots.
x[n] = x(n · T_s)
T_s = sampling period
f_s = 1 / T_s = sampling frequency

Sampling Rate & Aliasing

When f_s < 2 × F_max → high frequencies "fold back" into lower frequencies → distortion called **aliasing**
Prevention: Apply low-pass anti-aliasing filter before sampling (cutoff ≤ f_s/2)

Quantization Process

Mapping continuous amplitude values (after sampling) to a finite set of discrete levels.
Number of levels = 2^N
where N = number of bits (quantization resolution)
Example: 8-bit → 256 possible levels

Quantization Error

Difference between original sample value and nearest quantization level.
Behaves like added noise.
Max error ≈ ± (Δ / 2) where Δ = quantization step size
More bits → smaller Δ → smaller quantization error → better quality

Encoding Process

Each quantized level is assigned a unique binary code.
N-bit quantizer → 2^N possible binary codes
Most common: straight binary (natural binary code)
MSB = highest power of 2, LSB = lowest

Companding & Key Takeaways

Companding = Compression + Expansion
Logarithmic mapping → better resolution for small signals, coarser for large amplitudes
Main concepts recap:
  • Nyquist theorem: minimum sampling rate = 2 × highest frequency
  • Aliasing → distortion from undersampling
  • Quantization → approximation error (noise)
  • More bits = better precision, bigger data size
  • Encoding → binary representation of quantized values
  • Companding → efficient for wide dynamic range signals
